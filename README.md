# revanced-nomount

Inject the rvmm (ReVanced) modules using NoMount, eliminating traditional mounts (for [revanced-magisk-module](https://github.com/j-hc/revanced-magisk-module)).

## Usage
- Make sure the modules of whichever app you want is installed from [revanced-magisk-module](https://github.com/j-hc/revanced-magisk-module)
- Make sure you have [NoMount](https://github.com/maxsteeel/nomount) integrated into your kernel and its respective module installed.
- Flash [rvmm-nomount](https://github.com/maxsteeel/rvmm-nomount/releases) module and reboot.
